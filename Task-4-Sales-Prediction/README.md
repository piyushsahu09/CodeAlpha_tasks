# TASK 4: Sales Prediction using Python

## Objective
To predict future sales based on advertising spend, target segment, and platform
using regression techniques in Python.

## Steps Performed
- Data loading and cleaning
- Feature selection
- Train-test split
- Regression model training
- Sales prediction
- Model evaluation
- Data visualization

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Outcome
The model shows how advertising investment impacts sales and provides insights
that can help businesses optimize marketing strategies.
